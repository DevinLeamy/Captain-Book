### Tech

-   `React`: client
-   `Firebase`: client + server authentication
-   `Postgres`: database
-   `AWS S3`: image + file store
-   `Ktor`: async http server + http client
-   `Skrape{it}`: web scraping
-   `docker-compose`: deploying postgres
-   `nginx`: web server
